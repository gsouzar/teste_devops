## ▶️ Como Executar o Projeto

### 📋 Pré-requisitos

Antes de começar, você vai precisar ter instalado:

* Navegador

### 🔧 Instalação

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/teste_devops
```

2. Acesse a pasta do projeto:

```bash
cd teste_devops
```

4. Execute o projeto:

```bash
npm start teste_devops
```
